# 🖊️ ESP32 Auto Writer Pro
一个基于 **ESP32 + CoreXY** 的全功能自动写字机。
支持**中英文 + 标点符号 + 数字**，通过 **Web 上位机**控制。

> 🎯 目标：两个月内做出可直接使用的产品

---

## ✨ 功能特性
### ✅ 核心功能（MVP）
- [x] CoreXY 双轴运动控制（步进电机）
- [x] 舵机抬笔/落笔
- [x] 限位开关自动归零
- [x] 串口 / WebSerial 通信
- [x] **中文汉字**（浏览器端字体轮廓解析）
- [x] **英文、数字、标点符号**
- [x] 自定义字体选择
- [x] 字号、间距、行距调节
- [x] 笔画速度调节

### 🖥️ Web 上位机（Vue 3 + TypeScript）
- [x] 文字输入 + 实时预览
- [x] 字体选择（支持任意系统字体 / 自定义字体文件）
- [x] 参数调节（速度、抬笔高度、笔画粗细）
- [x] 串口连接管理
- [x] 发送 G-code 到设备
- [x] 绘制进度显示

## 🧠 系统架构
```
┌─────────────────────────────────────────────────┐
│ 浏览器 (Vue 3)                                   │
│                                                 │
│ 用户输入文字 → opentype.js 解析字体轮廓          │
│ → 提取笔画路径 → 生成 G-code                     │
│ → 通过 WebSerial API → 发送到 ESP32              │
└──────────────────────┬──────────────────────────┘
                       │
               WebSerial (USB / 蓝牙)
                       ▼
┌─────────────────────────────────────────────────┐
│ ESP32                                            │
│                                                 │
│ 串口接收 G-code → Grbl 运动规划                  │
│ → 步进电机驱动 (X/Y) → 舵机抬笔控制              │
│ → 限位开关检测                                   │
└─────────────────────────────────────────────────┘
```

## 🚀 快速开始
### 1️⃣ 硬件组装
1. 购买硬件（见硬件清单）
2. 组装 CoreXY 结构
3. 按接线图连接电路
4. 通电测试

### 2️⃣ 烧录固件
```bash
git clone https://github.com/your-username/esp32-auto-writer-pro
cd esp32-auto-writer-pro/firmware
# 使用 PlatformIO 打开并烧录
pio run --target upload
```

### 3️⃣ 启动 Web 上位机
```bash
cd web-ui
npm install
npm run dev
# 浏览器打开 http://localhost:5173
```

### 4️⃣ 连接设备
1. ESP32 通过 USB 连接电脑
2. 浏览器点击「连接设备」
3. 选择对应的串口
4. 输入文字 → 点击「开始绘制」
